# src/lib — Frontend utilities

Browser- + edge-side utilities. Imported by Astro pages and Svelte components.

- `api.js` — REST client wrapper for the SifterSearch API. ALL frontend → backend calls go through this. Handles auth header, base URL, error normalization. **Mega-file (1,391 lines).**
- `auth.svelte.js` — auth store (Svelte 5 rune state). Login state, JWT lifecycle.
- `pwa.svelte.js` — PWA install prompt + offline state.
- `usage.svelte.js` — usage / quota display state.
- `changelog.json` — generated changelog data (regenerated by pre-commit hook from `git log`). DO NOT edit manually.
- `imagekit.js` — image transform URL builder.
- `qrcode.js` — QR code rendering helper.
- `referral.js` — referral code helpers.
- `smart-quotes.js` — straight → curly quote pass for typography.
- `text-utils.js` — string helpers (slug, truncate, etc.).

## Subdirectory
- `stores/` — additional Svelte runes-based stores.
